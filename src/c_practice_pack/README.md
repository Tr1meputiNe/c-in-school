# C Language Practice Pack

This folder contains practice problems and reference answers for a C language study path.

## Structure

- `01_odd_even.md` to `12_student_average.md`: problem statements
- `answer/`: reference C solutions

## Suggested Order

1. 01-04: basics
2. 05-07: arrays and strings
3. 08-10: loops and algorithms
4. 11-12: pointers and structs

## Compile Example

```bash
gcc answer/01_odd_even.c -o odd_even
./odd_even
```
